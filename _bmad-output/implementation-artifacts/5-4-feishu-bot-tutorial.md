# Story 5.4: 飞书机器人添加教程页面

Status: ready-for-dev

## Story

As a 用户,
I want 查看如何在飞书中添加机器人的教程,
so that 可以在飞书中开始使用 OpenClaw。

## Acceptance Criteria

1. **AC1: 教程页面入口**
   - **Given** 实例已部署成功
   - **When** 用户访问实例详情页或帮助中心
   - **Then** 显示"如何在飞书中使用"教程入口
   - **And** 入口图标清晰可见
   - **And** 点击可打开教程页面

2. **AC2: 分步教程显示**
   - **Given** 用户打开教程页面
   - **When** 查看教程内容
   - **Then** 显示 3 个步骤的教程
   - **And** 每个步骤有清晰的标题和编号
   - **And** 步骤顺序明确（1/3, 2/3, 3/3）

3. **AC3: 步骤 1 - 搜索机器人**
   - **Given** 用户查看步骤 1
   - **When** 阅读步骤内容
   - **Then** 显示"在飞书搜索机器人名称"指引
   - **And** 显示机器人名称（来自飞书应用配置）
   - **And** 包含飞书搜索截图
   - **And** 高亮显示搜索框位置

4. **AC4: 步骤 2 - 添加机器人**
   - **Given** 用户查看步骤 2
   - **When** 阅读步骤内容
   - **Then** 显示"添加机器人到会话"指引
   - **And** 包含添加机器人的截图
   - **And** 说明单聊和群聊两种添加方式
   - **And** 高亮显示添加按钮位置

5. **AC5: 步骤 3 - 发送测试消息**
   - **Given** 用户查看步骤 3
   - **When** 阅读步骤内容
   - **Then** 显示"发送测试消息验证"指引
   - **And** 提供示例测试消息
   - **And** 说明预期回复
   - **And** 包含发送消息的截图

6. **AC6: 快捷按钮**
   - **Given** 教程页面已显示
   - **When** 用户需要快速操作
   - **Then** 提供"在飞书中打开"快捷按钮
   - **And** 点击按钮打开飞书应用
   - **And** 使用飞书协议链接（feishu://）

7. **AC7: 截图指引**
   - **Given** 每个步骤包含截图
   - **When** 查看截图
   - **Then** 截图清晰可读
   - **And** 关键区域有高亮标注
   - **And** 支持点击放大查看
   - **And** 图片加载有占位符

8. **AC8: 响应式设计**
   - **Given** 用户使用不同设备访问
   - **When** 在桌面、平板或手机查看
   - **Then** 教程布局自适应
   - **And** 移动端步骤可垂直滚动
   - **And** 图片在小屏幕上仍清晰

## Dev Notes

### 组件设计

**教程组件结构:**

```
FeishuBotTutorial.vue
├── StepIndicator         # 步骤指示器 (1/3, 2/3, 3/3)
├── TutorialStep          # 步骤容器
│   ├── StepTitle         # 步骤标题
│   ├── StepDescription   # 步骤描述
│   ├── StepImage         # 步骤截图（带高亮）
│   └── StepTip           # 步骤提示
├── QuickActionButton     # 快捷操作按钮
└── NavigationButtons     # 上一步/下一步按钮
```

### 视觉设计

**来自 UX 设计规范:**

- 主要按钮颜色: #1677FF (蓝色)
- 成功反馈: Message.success 3秒自动消失
- 卡片样式: Naive UI Card 组件
- 步骤指示器: 使用 Steps 组件

**布局要求:**
- 桌面端: 左右布局（步骤 | 图片）
- 移动端: 上下布局（步骤在上，图片在下）
- 最大宽度: 1200px
- 内边距: 24px

### 飞书协议链接

**打开飞书应用:**

```
// 打开飞书客户端
feishu://

// 打开飞书搜索
feishu://applink.feishu.cn/client/search/open

// 打开机器人对话
feishu://applink.feishu.cn/client/chat/open?id={chat_id}
```

- 标题: "搜索机器人"
- 描述: "在飞书客户端中搜索你的机器人名称"

**步骤 2: 添加机器人到会话**
- 标题: "添加机器人"
- 描述: "点击添加按钮，将机器人添加到单聊或群聊"
- 截图: 机器人详情页，高亮添加按钮
- 提示: "添加后即可在会话中与机器人对话"
